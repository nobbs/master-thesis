function colvec F = SolverNodal.spacetimeLoadVector ( )

Assemble the load vector for the space time system.

This is done by using numerical quadrature for the spatial parts of the source term and the initial condition. Additionaly we use a simple trapezoidal quadrature for the temporal part of the source term.
