Cumulative trapezoidal quadrature.

Evaluates the integral through a cummulative trapezoidal quadrature on a equidistant grid. If is a periodic function and the interval of integration is a multiple of the period of , then is_periodic should be set to true and grid and function values should be given for the half open interval .
